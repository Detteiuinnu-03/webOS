How to upgrade to Terbium v2.0

For Site Owners/Administrators:

1: Clone/Download the new Repository

2: Download the dependencies using npm (preferebly use pnpm)

3: If your port forwarding setup already has Port 6969, Simply go to the .env file and change PORT=3000 to PORT=6969. If you are new to setting this up ignore this

4: Start the HTTP Server via npm start (once again, its recommended to use pnpm)


Bonus: If you are deploying this to a link bot Its recomended to use MASQR. If you dont know what it is find out more about it Here

For Normal Proxy Users:

Unfortunatly there isnt much you can do. You have to wait for your site administrator to update the site. However you can request new links in the TN Discord that will already be updated to the new version
